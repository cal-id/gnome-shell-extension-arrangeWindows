# gnome-shell-extension-arrangeWindows

Arrange windows on the monitor.

#### Hot keys

* `Ctrl+Alt+1` Cascade windows
* `Ctrl+Alt+2` Tiling windows
* `Ctrl+Alt+3` Side by side windows
* `Ctrl+Alt+4` Stack windows

## Install

```
make install
```
restart system
